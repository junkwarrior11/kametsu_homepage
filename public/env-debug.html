📊 診断結果サマリー

上記のテストボタンをクリックして診断を開始してください。

⚠️ エラーが出る場合

  1. 404エラー（Not Found）: Supabaseにテーブルが存在しません
    • Supabase Dashboard → Table Editor でテーブルを作成
  2. 500エラー（Internal Server Error）: 環境変数が未設定またはSupabase接続エラー
    • Netlify → Environment variables で SUPABASE_ANON_KEY を確認
    • Supabase → Settings → API で anon key を再取得
  3. 認証エラー: SUPABASE_ANON_KEY が間違っている
    • Supabaseで正しいキーをコピーして再設定
